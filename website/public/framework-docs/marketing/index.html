    Cross-Platform References¶

    Don't: - Cross-promote on Hacker News (against culture) - Spam same content across all platforms simultaneously - Reference other platform discussions on Reddit (feels promotional)

    Do: - Share Twitter thread on LinkedIn - Reference LinkedIn discussion in Twitter replies - Link to GitHub from everywhere - Share demo results and feedback across platforms - Thank community on all platforms
