### Icon Specifications
- Format: SVG (vector)
- Base size: 24x24px
- Style: Lucide icons (consistent stroke width)
- Color: Uses `currentColor` (inherits from parent)

## Usage in InDesign

### For the Adobe Stock image:
1. Place at full resolution for print (300 DPI)
2. Can be used with overlay/blend modes
3. Works well with white or light text overlay

### For SVG icons:
1. Import as vector graphics to maintain sharpness at any size
2. Can recolor by editing the stroke attribute
3. Recommended sizes for print: 0.5" - 1" for inline, 2"+ for feature callouts
