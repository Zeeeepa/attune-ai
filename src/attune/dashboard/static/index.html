Dashboard Help

The Agent Coordination Dashboard monitors multi-agent systems and human-in-the-loop workflows.

Requires Redis to store and retrieve agent coordination data.

  • Multi-Agent Coordination - Track multiple agents working together
  • Approval Gates - Approve/reject agent actions in real-time
  • Quality Monitoring - Track quality metrics across tiers
  • Debugging - Troubleshoot agent communication issues
  • Running basic workflows (code-review, test-gen, etc.)
  • Single-agent workflows
  • No human approval required
  • Not monitoring quality metrics

Most users can run workflows without ever opening this dashboard.

Active Agents - Currently running agents with progress bars

Approval Requests - Pending actions requiring human approval

Quality Metrics - Performance tracking by workflow/stage/tier

Recent Signals - Agent-to-agent coordination messages

Event Stream - Real-time event log from all agents

Underperforming Stages - Stages below quality threshold

Docker:

docker run -d -p 6379:6379 redis:latest

macOS (Homebrew):

brew install redis && brew services start redis

Linux (apt):

sudo apt install redis-server && sudo systemctl start redis
