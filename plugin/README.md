# attune-ai Plugin

Cowork plugin for AI-powered developer workflows.

## Installation

Install from the plugin directory or marketplace.

## Skills

| Skill | Description |
|-------|-------------|
| `security-audit` | Vulnerability scanning |
| `code-review` | Automated code review |
| `test-gen` | Test generation |
| `bug-predict` | Bug prediction |
| `perf-audit` | Performance analysis |
| `release-prep` | Release preparation |

## Commands

- `/workflows` - Run automated analysis workflows

## MCP Tools

When connected, these tools are available:
- `security_audit`
- `code_review`
- `test_generation`
- `bug_predict`
- `performance_audit`
- `release_prep`

## Requirements

- Python 3.10+
- `pip install attune-ai`
